## What I Learned About Software Engineering

Before this class, I thought software engineering was mostly just writing code and making programs work. Since we used tools for building web applications, it would be easy to think the class was only about making websites. But I learned that web development was just the way we practiced bigger software engineering ideas. One of those ideas was **configuration management**, which means keeping track of a project’s files, code, settings, versions, and changes. Tools like Git and GitHub help with this by showing what changed, who changed it, and how to go back if something breaks. This matters for web apps, but it also matters for games, mobile apps, school projects, and anything else where people need to stay organized.

Another important thing I learned was **agile project management**, which is a way of working on projects in smaller steps instead of trying to plan everything perfectly at the start. In this class, we used **Issue Driven Project Management**, where the work is split into issues, or small tasks, bugs, features, and improvements. This made the project feel less overwhelming because instead of just saying “finish the app,” we could focus on clear tasks like fixing a page or adding a feature. I could see myself using this outside of web development too, like for a group presentation, event planning, or any project where people need to know what has to be done and who is doing it.

I also learned about **ethics in software engineering**, which means thinking about how software affects real people. Software is not just about whether it works; it can affect privacy, safety, fairness, and trust. For example, if an app collects user data, the developers need to think about how that data is protected. This applies to web apps, but also to software used in hospitals, banks, cars, schools, and many other areas. Overall, this class taught me that software engineering is not just coding. It is also about organization, teamwork, responsibility, and building things in a careful and professional way.
